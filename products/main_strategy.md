# メイン戦略（autoT）

Threads 市場観測の最上位方針。`t_insight.md` はこのファイルに合致する投稿・話題だけを書く。

## いまの段階

**観測フェーズ** — 公開投稿から、見込み客・信頼に関係する話題の型を集める。投稿自動化は別フェーズ。

## 観測の目的

- 転職・キャリア・リモート・独立・エンジニアのキャリア階段に関する **話題の型** を把握する
- いいね数の大小で優劣をつけない（Threads API では他人投稿の metrics は取れない前提）
- メイン戦略に合致しない話題は観測に載せない

## 合致するテーマの例

転職、キャリアチェンジ、フルリモート、エージェント、独立、副業、製造業エンジニア、育児と仕事、スキル転換

## 合致しない例

ゲーム、政治、投資のみ、AI ツールの機能比較だけ、戦略外ジャンル
