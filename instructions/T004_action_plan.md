# T004_action_plan — 対応方針

## 目的

承認された仮説に基づき、対応方針を `action_plan.md` にまとめる。

## 入力

- `./incident_context.md`（必須）
- `./hypothesis.md`（必須）

## 作業

1. `incident_context.md` と `hypothesis.md` を読む。
2. `../instructions/format.action_plan.md` を読み、出力形式を把握する。
3. 手順1の内容を根拠に、`format.action_plan.md` に従い `action_plan.md` を作成する。
4. 変更してよいのは `action_plan.md` のみ。

## 停止条件

- `hypothesis.md` が存在しない、または仮説が空 → `abort.txt` に理由を書いて終了。
